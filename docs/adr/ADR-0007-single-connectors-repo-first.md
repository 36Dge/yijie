# ADR-0007: MVP 阶段使用单一 Connectors 仓库

## 状态

Accepted

## 日期

2026-07-04

## 背景

易界 AI 将接入 Amazon、Temu、Shopee、TikTok Shop 以及物流、邮箱、飞书、钉钉等外部平台。

## 决策

MVP 阶段所有连接器统一放在 `yijie-connectors`，按平台和三方应用模块隔离。

## 备选方案

每个平台独立仓库可以更细粒度管理发布，但早期会放大工程复杂度。

## 影响

- token vault、限流、重试、审计和 MCP 工具执行统一治理；
- 后续按平台复杂度和团队规模再拆仓。

## 风险

仓库可能变大，需要模块约束和清晰目录。

## 后续动作

- [ ] 定义连接器模块规范；
- [ ] 定义 MCP 工具执行契约。

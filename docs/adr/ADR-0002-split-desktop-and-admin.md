# ADR-0002: 拆分桌面端和 Admin 后台仓库

## 状态

Accepted

## 日期

2026-07-04

## 背景

易界 AI 面向卖家的主产品是 Mac 桌面端，同时需要内部 Admin Web 管理租户、权限、插件、知识库和审计。

## 决策

桌面端使用 `yijie-desktop`，Admin 后台使用 `yijie-admin-web`，两个产品独立仓库、独立发布。

## 备选方案

放入同一仓库可以短期减少配置，但会混淆权限、发布节奏和产品体验。

## 影响

- 桌面端专注卖家 Chat、任务和 sidecar；
- Admin 专注内部治理、审计和运营；
- 共享类型通过 `yijie-contracts` 生成。

## 风险

公共组件可能重复，后续如重复明显再考虑 `yijie-ui-kit`。

## 后续动作

- [ ] 定义共享契约生成流程。
